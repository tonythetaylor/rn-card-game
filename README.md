# Expo Swipe Cards

This is a basic project for studying purposes. It uses Expo and react native animations.
It shows a deck of cards like Tinder App.

## Running the App

### Install Expo CLI
```sh
$ yarn global add expo-cli
```

### Install dependencies
```sh
$ yarn
```

### Start the project
```sh
$ yarn start
```
